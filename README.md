# SkyTeams Internship 2025 Fall – Rahimah Ansari

## Overview

This repository contains all my programming exercises and solutions for the SkyTeams Internship 2025 Fall.  
Each week has its own folder with solutions file.

Each week folder contains:

- Solution files for each question
- Corresponding test files
- A weekly README.md summarizing my learning

## Week 1 – Fundamentals, Testing, and GitHub Basics

**Topics Covered:**

- Variables, Data Types, Loops, Conditionals, Functions
- Arrays, Objects, Clean Code
- Git & GitHub basics (clone, branch, commit, push, pull request)
- Basic testing principles (unit tests for functions)

**Exercises:**

1. Swap the values of two variables
2. Check if a number is even, odd, or prime
3. Sum all numbers in an array
4. Write a function to reverse a string/list
5. Find the largest number in a list or collection
6. Refactor messy code into readable functions
